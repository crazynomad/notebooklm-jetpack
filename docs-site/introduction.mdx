# 给 NotebookLM 装上喷射背包 🚀

NotebookLM 是最好的 AI 知识工具，但「把内容喂进去」这一步太痛苦了。

**Jetpack 解决这个问题。**

<CardGroup cols={3}>
  <Card title="一键导入" icon="bolt">
    网页、Substack、微信公众号、文档站、播客、AI 对话 — 粘贴链接即可导入
  </Card>
  <Card title="突破来源限制" icon="layer-group">
    将 20 篇文章聚合为 1 个 PDF，只占 1 个来源配额
  </Card>
  <Card title="智能抢救" icon="wrench">
    自动检测导入失败的来源，一键批量修复
  </Card>
</CardGroup>

## 支持导入的内容

| 类型 | 来源 |
|---|---|
| 🌐 网页 | 任意网页、Substack、微信公众号 |
| 📚 文档站 | Docusaurus、VitePress、MkDocs、GitBook、Mintlify 等 14+ 框架 |
| 🎙️ 播客 | Apple Podcasts、小宇宙 |
| 🤖 AI 对话 | Claude、ChatGPT、Gemini |
| 📡 RSS | 任意 RSS/Atom 源 |

## 安全与隐私

- ✅ 完全免费，无需注册登录
- ✅ 纯客户端运行，不上传任何数据
- ✅ [开源代码](https://github.com/crazynomad/notebooklm-jetpack)，可审计
- ✅ 中英双语界面，自动适配
