## 导入你的第一个网页

<Steps>
  <Step title="打开 NotebookLM 笔记本">
    在浏览器中打开 [NotebookLM](https://notebooklm.google.com)，创建或进入一个笔记本。

    <Warning>
      请确保打开的是**笔记本内部页面**（不是首页），否则导入会失败。
    </Warning>
  </Step>
  <Step title="浏览你想导入的网页">
    在另一个标签页中打开你想导入的文章（比如一篇博客文章）。
  </Step>
  <Step title="点击 Jetpack 图标，导入">
    1. 点击工具栏上的 Jetpack 图标
    2. 点击「已收藏」或直接在输入框粘贴 URL
    3. 点击「导入」按钮
    4. 等待几秒，内容会自动出现在 NotebookLM 的来源列表中
  </Step>
</Steps>

<Tip>
  **更快的方式**：在任意网页上右键 → 「导入到 NotebookLM」，无需打开扩展面板。
</Tip>
