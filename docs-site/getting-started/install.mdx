## 从 GitHub 下载安装

目前推荐的安装方式——直接从 GitHub Release 下载最新版本。

<Steps>
  <Step title="下载扩展包">
    前往 [GitHub Releases](https://github.com/crazynomad/notebooklm-jetpack/releases/latest)，下载 `.zip` 文件
  </Step>
  <Step title="解压文件">
    将下载的 `.zip` 文件解压到一个固定的目录（不要删除，Chrome 需要持续读取该目录）
  </Step>
  <Step title="加载到 Chrome">
    1. 打开 `chrome://extensions`
    2. 开启右上角「开发者模式」
    3. 点击「加载已解压的扩展程序」
    4. 选择解压后的目录
  </Step>
  <Step title="固定到工具栏">
    点击 Chrome 右上角拼图图标 🧩 → 找到 NotebookLM Jetpack → 点击 📌 固定
  </Step>
</Steps>

<Tip>
  更新时重复上述步骤，覆盖解压目录后在 `chrome://extensions` 页面点击刷新按钮即可。
</Tip>

## Chrome Web Store

<Info>
  扩展已提交 Chrome Web Store 审核，审核通过后将更新此处链接，届时可一键安装并自动更新。
</Info>

## 从源码构建

如果你想使用最新开发版或参与开发：

<Steps>
  <Step title="克隆仓库">
    ```bash
    git clone https://github.com/crazynomad/notebooklm-jetpack.git
    cd notebooklm-jetpack
    pnpm install
    pnpm build
    ```
  </Step>
  <Step title="加载到 Chrome">
    1. 打开 `chrome://extensions`
    2. 开启右上角「开发者模式」
    3. 点击「加载已解压的扩展程序」
    4. 选择项目中的 `dist/chrome-mv3` 目录
  </Step>
</Steps>
