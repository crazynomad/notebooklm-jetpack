## 三种导入模式

扩展提供三种网页导入方式，适用于不同场景。

### 单个导入

最简单的方式 — 粘贴一个 URL 即可。

<Steps>
  <Step title="打开 Jetpack 面板">
    确保已打开一个 NotebookLM 笔记本，然后点击 Jetpack 图标。
  </Step>
  <Step title="输入 URL">
    在输入框中粘贴网页链接，或点击「当前标签页」快捷按钮直接导入当前页面。
  </Step>
  <Step title="点击导入">
    扩展会自动提取正文内容并导入到 NotebookLM。
  </Step>
</Steps>

<Tip>
  **更快捷**：在任意网页右键 →「导入到 NotebookLM」，一步完成。
</Tip>

### 批量导入

一次导入多个 URL。

1. 切换到「批量」模式
2. 每行粘贴一个 URL（或用逗号分隔）
3. 点击「批量导入」

<Note>
  也可以点击「导入所有打开的标签页」，自动获取当前所有标签页的 URL 并填入列表。
</Note>

### 右键菜单

浏览任意网页时：
- **右键页面空白处** →「导入到 NotebookLM」→ 导入当前页面
- **右键链接** →「导入到 NotebookLM」→ 导入链接目标页面

## 导入失败？

如果某些 URL 导入失败，Jetpack 会显示失败列表，支持：
- **重试失败** — 一键重试所有失败项
- **逐个重试** — 点击单个失败项旁的重试按钮

<Warning>
  导入前请确保已打开一个 NotebookLM **笔记本页面**（不是首页）。
</Warning>
